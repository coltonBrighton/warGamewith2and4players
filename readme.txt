This project uses HTML, Bootstrap, and Javascript to simulate the card game war for either 2 players or 4 players and outputs results of each turn to a table.
All you have to do is click a button ether 2 players or 4 players and the simulation will run.
================================================================================================
Installation: 
  1) select the code dropdown menu
  2) select download ZIP
  3) run npm init and npm install in termainal to download bootstrap
